# SonicLab Flow

🎵 专业音效管理与播放工具，专为视频剪辑师、音频设计师打造

## 🚀 功能特性

### 核心功能
- 📂 **智能音效库管理** - 支持多级文件夹结构，快速浏览海量音效
- 🎧 **波形可视化播放** - 直观的波形图显示，精准定位播放位置
- ⏯️ **流畅播放控制** - 播放、暂停、跳转、框选即播，专业级操作体验
- 🏷️ **标签管理系统** - 自定义彩色标签，灵活分类管理音效
- 🤖 **AI 智能标签** - 自动翻译中文名称，智能生成音效描述
- 🎨 **UI 个性化** - 主题色相、饱和度、亮度、对比度自由调节

### v0.2.3 最新更新
- ✨ 新增 UI 界面设置，支持主题色相、饱和度、亮度、对比度和波形颜色个性化调节
- ✨ 新增波形未选中亮度调节
- 🎯 优化播放高光尾迹显示
- 🐛 修复 UI 调节闪退问题
- 🐛 修复后台扫描回调问题

## 🎯 快速开始

1. **导入音效库** - 点击「打开文件夹」选择音效目录
2. **播放音效** - 双击或按空格键播放
3. **添加标签** - 右键音效添加标签分类
4. **AI 分析** - 使用 AI 功能自动生成中文名称和描述

## 🛠️ 系统要求

- **macOS**: 10.15 或更高版本
- **Windows**: Windows 10 或更高版本
- **内存**: 建议 4GB 以上
- **磁盘**: 根据音效库大小而定
